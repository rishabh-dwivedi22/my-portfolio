# My Personal Portfolio 

Welcome to my personal portfolio repository.
##  Tech Stack
This portfolio is built using modern web technologies to ensure a fast and responsive experience:
* **Frontend:** HTML5, CSS3, JavaScript (ES6+)
* **Styling:** SCSS (Sass) & Bootstrap 4/5
* **Tools:** Prepros for compilation, VS Code, and Git for version control.

##  Key Features
- **Fully Responsive:** Works perfectly on desktops, tablets, and mobile devices.
- **Clean UI/UX:** Focused on readability and professional aesthetics.
- **Project Showcase:** Highlights my key works, including **InterviewSaathi** and other web-based tools.
- **Optimized Performance:** Minimized CSS/JS and organized folder structure for faster loading.

##  Folder Structure
- `/css`: Compiled stylesheets.
- `/scss`: Source Sass files for better styling management.
- `/js`: Interactive elements and logic.
- `/images`: Optimized assets and icons.
- `/lib`: External libraries and dependencies.
